# requirements-deploy.txt — SLIM dependencies for cloud demo deploy (Render free tier)
#
# The full requirements.txt pulls in torch, tensorflow, ultralytics, mediapipe,
# deepface, chromadb, etc. (~4-5 GB) which won't fit a free-tier build/runtime.
#
# The cloud DEMO runs on synthetic data and never executes live inference, so the
# heavy ML stack is not needed. All ML imports in the codebase are lazy (inside the
# pipeline thread / specific handlers), so the API starts fine without them. Features
# that need the ML stack (live chat RAG, SHAP, agent) degrade gracefully.
#
# Use this for the cloud demo:  pip install -r requirements-deploy.txt
# For full local/ML use:        pip install -r requirements.txt
